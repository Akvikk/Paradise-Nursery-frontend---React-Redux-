**Final Project**: Paradise Nursery is the frontend of a houseplant shopping web application built as my final assignment.

## Short Info
- **Purpose**: Demonstrate a full-featured frontend with product browsing, cart management, and checkout placeholder to satisfy the course rubric and peer-reviewed requirements.
- **Tech**: React 18, Vite, Redux Toolkit, React Router.
- **Pages**:
  - Landing page with background and company info
  - Product listing (6 products across 3 categories)
  - Shopping cart with quantity controls, delete, totals, and checkout placeholder
- **State**: Redux Toolkit stores cart state; persisted to `localStorage`.

## Run locally
1. `npm install`
2. `npm run dev`

## Deploy
Deploy the built `dist` folder to GitHub Pages, Vercel, or Netlify. For GitHub Pages, name your repo `paradise-nursery` and enable Pages from the `main` branch.
